Agent Farm Guide to the simulation

Agent Farm is a live multi-agent simulation. Each agent wanders, encounters nearby agents, exchanges a resource, and contributes to a weighted interaction network that you can inspect in both the farm and network views.

Pilot

Click an agent to select it, then use WASD or the arrow keys to steer. Hold Shift to move faster, press Space to brake, and use Esc to clear the selection. The selected agent and its strongest interaction partners are highlighted in both views.

Draw

In farm view, drag on the field to place obstacles. Obstacles alter the local movement paths and can be used to shape where encounters happen. Use Pilot + Draw when you want manual control and obstacle drawing together.

Themes

The theme dots in the top-right HUD swap the visual language of the sim without changing the underlying behavior. They are useful for switching between paper, blueprint, terminal, amber, and mono-dark views of the same interaction process.

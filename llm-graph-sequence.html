Sequential graph generation

Watch a model emit edges against a fixed ground-truth graph.

This viewer uses actual Game of Thrones Season 1 LLM outputs. Ground-truth edges sit underneath in soft grey; emitted edges are highlighted as correct or hallucinated as the sequence unfolds.

Current Scope

Run-by-run playback

Switch prompt order and model run, then compare the emitted sequence to the fixed reference graph.

Color Logic

Grey, green, red

Grey shows the base graph, green marks emitted edges that exist in ground truth, and red marks hallucinated edges.
